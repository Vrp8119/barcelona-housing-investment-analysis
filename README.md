# Barcelona Housing Investment Analysis

Predicting sale and rental prices in Barcelona's real estate market 
using ensemble ML models, and identifying rental investment opportunities.

## What this project does
- Builds regression models (Bagging, Random Forest, XGBoost) for sale and rental prices
- Applies the rental model to for-sale listings to estimate ROI and payback period
- Identifies properties with ≥10% annual return and ≤5 year payback period

## Key findings
- Random Forest outperformed other models in both stability and accuracy
- Barcelona's best rental investments are small, old, no-frills flats under €80,000
- Median payback period across the market: ~19.5 years

## Stack
Python · Pandas · Scikit-learn · XGBoost · Matplotlib · Seaborn

## Data
Real Barcelona housing data provided as part of a university master's program.
